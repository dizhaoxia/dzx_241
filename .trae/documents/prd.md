## 1. 产品概述

图片标注工具是一个基于 Web 的轻量级图片标注应用，旨在为用户提供快速、直观的图片标注能力。用户可以上传图片后在画布上绘制矩形框、多边形区域和添加文本标签，并将标注数据导出为 JSON 文件进行持久化存储。

- 目标用户：数据标注人员、AI训练数据准备者、图片审核员
- 核心价值：零安装、零后端依赖、开箱即用的图片标注体验

## 2. 核心功能

### 2.1 功能模块

1. **标注画布页面**：图片上传、标注工具栏、画布操作、数据管理

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 标注画布页面 | 图片上传 | 点击按钮上传 jpg/png 图片，上传后自动适配画布大小 |
| 标注画布页面 | 矩形绘制工具 | 鼠标拖拽生成矩形标注框 |
| 标注画布页面 | 多边形绘制工具 | 鼠标逐点点击绘制多边形，双击完成绘制 |
| 标注画布页面 | 文本添加工具 | 点击标注框后输入文本标签 |
| 标注画布页面 | 基础编辑 | 拖拽移动标注框、拖拽控制点调整大小、Delete键删除选中标注、点击选中高亮 |
| 标注画布页面 | 数据管理 | 导出JSON、导入JSON、清空所有标注 |
| 标注画布页面 | 画布操作 | 清空画布、鼠标滚轮缩放、按住空格+鼠标拖拽平移 |

## 3. 核心流程

用户打开应用 → 上传图片 → 图片自动适配画布 → 选择标注工具（矩形/多边形/文本）→ 在画布上进行标注 → 编辑标注（移动/调整/删除）→ 导出标注数据为JSON / 从JSON导入恢复标注

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深灰蓝（#1e293b）为底，亮青色（#06b6d4）作为强调色
- 辅助色：琥珀色（#f59e0b）用于警告操作，翡翠绿（#10b981）用于成功反馈
- 按钮风格：圆角微凸起，hover 时轻微上浮带阴影
- 字体：使用 JetBrains Mono 作为工具栏标识字体，Source Han Sans / Noto Sans SC 作为界面字体
- 布局风格：左侧工具栏 + 中央画布 + 顶部操作栏的经典工具布局
- 图标风格：线性图标，2px 描边

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 标注画布页面 | 顶部操作栏 | 图片上传按钮、导出JSON按钮、导入JSON按钮、清空按钮，深色背景+亮色图标 |
| 标注画布页面 | 左侧工具栏 | 选择工具、矩形工具、多边形工具、文本工具，竖向排列，选中态高亮 |
| 标注画布页面 | 中央画布区 | 白色/棋盘格背景、Fabric.js画布，底部缩放比例显示 |
| 标注画布页面 | 右侧属性面板 | 显示选中标注的属性信息（类型、坐标、尺寸） |

### 4.3 响应式设计

- 桌面优先设计，画布区域自适应窗口大小
- 最小支持宽度 1024px
- 工具栏固定宽度，画布区弹性伸缩
